# Okiba

This repository contains okiba bin's front-end code. The web app is using codemirror editor, react library with vite bundler.

## Usage

Add server URL in `.env.local` with `VITE_SERVER_URL` key.

```shell
// get deps
$ pnpm install

// build
$ pnpm run build

// dev server
$ pnpm run dev

// preview
$ pnpm run preview
```
